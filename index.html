FAQ

よくある質問

Airbnb以外の予約サイトでも使えますか？ +
現在はAirbnb専用です。Booking.comやExpediaなどのOTAへの対応は今後検討しています。
AIはどのくらいの質問に答えられますか？ +
物件情報やFAQを事前に学習させることで、一般的なゲストからの質問の約80〜90%に自動対応できます。答えられない質問はLINEで通知されるので、放置になることはありません。
AIプランからおもてなしAI+への変更はできますか？ +
はい、税抜1万円の追加料金（初期費用）でアップグレード可能です。
最低契約期間はありますか？ +
AIコンシェルジュは3ヶ月〜、おもてなしAI+は6ヶ月〜の契約となります。AIの学習・調整に約1ヶ月、安定運用で効果を実感いただくために最低期間を設けています。
月額にはどこまでの対応が含まれますか？ +
AIシステムの24時間稼働・スプレッドシート記録・LINE通知の維持に加え、Q&Aの追加や回答修正（AIコンシェルジュ：月3件まで / おもてなしAI+：月10件まで）が含まれます。超過分は1件1,000円で対応可能です。
複数物件をまとめて依頼できますか？ +
はい、可能です。5件以上の場合は割引料金をご用意していますのでお気軽にご相談ください。
スプレッドシートはGoogleスプレッドシートですか？ +
はい、Googleスプレッドシートでリアルタイムに共有します。スマートフォンからでも確認可能です。
途中解約はできますか？ +
最低契約期間内の解約の場合、残月分の月額料金をお支払いいただきます。最低期間終了後は、1ヶ月前のご連絡でいつでも解約可能です。
AIの回答精度はどうやって上がりますか？ +
導入後、スプレッドシートで「AIが答えられなかった質問」を確認できます。それをもとにQ&Aを追加していくことで、物件ごとにAIの精度が向上していきます。多くの物件で、1〜2ヶ月後には対応率90%以上に達します。
他の民泊管理サービスと併用できますか？ +
はい、併用可能です。AIコンシェルジュはお部屋にQRコードを設置する独立したサービスのため、既存の清掃代行や運営代行サービスとの併用に支障はありません。
